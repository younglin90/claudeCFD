# Group 6: 인공 확산 / 운동론적 플럭스 방법
# (Artificial Diffusivity and Kinetic Flux Methods)

## 핵심 아이디어

Abgrall 진동을 **수치 확산을 통해 억제**하는 접근법:

1. **인공 질량 확산 (AMD)**: 밀도 불연속부에 선택적으로 확산을 추가하되, 운동량/에너지 방정식과 **일관되게 결합**하여 Interface Equilibrium Condition(IEC)을 만족시킴
2. **운동론적 플럭스 (Kinetic Scheme)**: BGK 볼츠만 운동론에서 유도된 플럭스로 수치 확산 계수 `λ`를 동적으로 조절하여 정상 접촉 불연속에서 정확한 포획 달성

## Paper A: Jain, Agrawal, Moin — Stanford CTR (2023) — 국소화 인공 확산 (LAD)

**파일**: `2307.03257v1.pdf`
**제목**: "Stable, entropy-consistent, and localized artificial-diffusivity method for capturing discontinuities"
**저널**: arXiv:2307.03257v1 (2023)

### 방법: LAD (Localized Artificial Diffusivity)

단일 성분 기반 LAD를 다성분/2상 흐름으로 확장하면서 IEC를 만족시키는 AMD 구성.

**핵심 발견**: AMD(Artificial Mass Diffusivity)는 IEC를 만족하지만 ATD(Artificial Thermal Diffusivity)는 만족하지 않음 → 5방정식 모델에서 AMD가 Abgrall 진동 억제에 적합.

**인공 확산 항 (운동량, 에너지와 일관된 결합)**:

```
질량:    A_ρ    = ∂_j(D*·∂_j ρ)
운동량:  A_{ρu} = ∂_j(D*·u_i·∂_j ρ) + ∂_j(β*·∂_k u_k·δ_{ij})
에너지:  A_E    = ∂_j(D*·∂_j ρ·u_k u_k/2)
               + ∂_j(D*·∂_j(ρe))
               + ∂_j(β*·∂_k u_k·δ_{ij}·u_i)
```

이 결합이 IEC를 보장하는 이유: 균일한 `(p, u)` 상태에서 `A_ρ`, `A_{ρu}`, `A_E` 기여가 상쇄되어 `∂p/∂t = 0` 달성.

**AMD 계수** `D*` (Eq. 14):
```
D* = C_D · (1/ρ) · [Σ_j |∂^r ρ/∂x^r_j|·(Δx_j)^{r+1}] · (|u_j| + c_s) · f_D
```

**접촉 불연속 감지 함수** `f_D` (Eq. 17) — 핵심 기여:
```
f_D = [|∂_j ρ|² / (|∂_j ρ|² + a·(θ² + ω_i·ω_i))] × (ρ / (|u|² + ε))

θ = ∇·u  [팽창률]
ω_i = (∇×u)_i  [와도]
```
→ 충격파(큰 θ)와 소용돌이(큰 ω)에서는 비활성화, 순수 접촉 불연속에서만 활성화.

**ABV (Artificial Bulk Viscosity) 계수** `β*` (Eq. 15):
```
β* = C_β·ρ·[Σ_j |∂^r θ/∂x^r_j|·(Δx_j)^{r+2}]·H(-θ)·f_β
f_β = θ² / (θ² + a·ω_i·ω_i + ε)  [충격파 감지]
```

**2상 흐름 확장** (Section IIF):

밀도 구배를 체적분율 가중 상 밀도 구배로 대체:
```
∂_j ρ → Σ_l ϕ_l·∂_j ρ_l
```
→ 확산 계면(ACDI) 인터페이스에서 AMD가 자연 인터페이스 정규화와 충돌하지 않음.

IEC 달성: 5방정식 모델에서 AMD 사용 시 IEC 자동 만족 (ATD와 달리).

### 수치 스킴

- 2차 중심 유한차분 (FDM)
- 명시적 시간 적분 (Euler + RK)
- WENO/flux limiter 없음 — AMD/ABV가 유일한 안정화 메커니즘
- Dynamic Smagorinsky SGS 모델 (LES)

### 장단점

| | |
|---|---|
| ✅ | IEC 만족 (AMD 기반) |
| ✅ | 물리적으로 일관된 인공 확산 (엔트로피 일관성 있음) |
| ✅ | 기존 중심 차분 코드에 소규모 추가로 구현 가능 |
| ✅ | 접촉 불연속 선택적 활성화 → 충격파/와도에서 비활성화 |
| ⚠ | **진동 억제가 확산에 의존** → 계면 스미어링(smearing) 부작용 |
| ⚠ | Stiffened gas EOS — 실제유체(cubic EOS) 직접 미적용 |
| ❌ | 완전 보존형이나, 진동 억제 성능은 PEP 계열보다 낮음 |

## Paper B: Roy & Rao (2025) — 운동론적 양성 보존 스킴 (동적 λ)

**파일**: `[적용해볼것] lambda로 수치확산계수.pdf`
**제목**: "A Kinetic Scheme Based On Positivity Preservation For Multi-component Euler Equations"
**출처**: arXiv:2411.00285v2 (2025)

### 방법: 동적 λ 운동론적 스킴

BGK 볼츠만 방정식에서 유도된 수치 플럭스:

```
G_{j+1/2} = ½(G_j + G_{j+1}) - (λ_{j+1/2}/2)·(U_{j+1} - U_j)

λ_{j+1/2}: 수치 확산 계수 (단순 특성 속도 기반 추정치)
```

**양성 보존 조건으로부터 λ 하한 결정**:
```
λ_{j+1/2} ≥ max(
  -u_j + sqrt((γ_j-1)/(2γ_j))·a_j,
  u_{j+1} + sqrt((γ_{j+1}-1)/(2γ_{j+1}))·a_{j+1}
)
```

**Abgrall 문제 해결: 정상 접촉에서 λ → 0**

정상 접촉 불연속에서는 `ΔG = u·ΔU` (플럭스 점프 = 속도 × 상태 점프). 이를 이용한 적응형 λ (Rankine-Hugoniot 기반):
```
λ_RH = min_i(|ΔG_i| / (|ΔU_i| + ε₀))

진정한 정상 접촉이라면 λ_RH = u → 인터페이스 속도와 일치
→ 허위 수치 확산 없이 접촉 불연속 정확히 포획
```

**3차 정확도 확장**:
Chakravarthy-Osher 플럭스 제한자 + minmod, 양의/음의 운동론적 플럭스 성분 분리 적용.

**2D 확장**: 3-속도 모델 (`λ₁, λ₂, -λ₁-λ₂`)

### 다성분 처리

혼합 γ 규칙:
```
γ_mix = Σ_i Y_i·γ_i / Σ_i Y_i/(γ_i-1)  [질량분율 가중 혼합]
```
→ 이상기체 전용.

### 장단점

| | |
|---|---|
| ✅ | 완전 보존형 |
| ✅ | 양성 보존 (밀도/압력 양수 보장) |
| ✅ | 정상 접촉 불연속 정확 포획 (λ → 0) |
| ✅ | 구현 단순 — Lax-Friedrichs 타입 플럭스에 동적 λ 추가 |
| ❌ | **이상기체 전용** — 실제유체(cubic EOS) 없음 |
| ❌ | 다성분 이상기체에 한정 |
| ⚠ | λ_RH 계산에서 분모가 0이 될 수 있어 ε₀ 튜닝 필요 |

## 사용자 코드 적용 권장

두 방법 모두 실제유체(real gas / transcritical) 직접 지원이 없어 현재 코드 요구사항에 **2차 적합**.

단, 다음 상황에서 유용:

- **Jain LAD**: 기존 중심차분(FDM/FVM) 코드에 빠르게 압력 진동 억제 효과를 추가하고 싶을 때 → **프로토타입/검증용으로 적합**. IEC 만족 조건은 SRK/PR EOS로 확장 가능(부분 밀도 구배 표현 수정 필요).

- **Roy & Rao λ**: Godunov 타입 FVM에서 Lax-Friedrichs 계열 플럭스를 이미 사용 중이라면 λ 동적 조정만으로 접촉 불연속 포획 개선 가능 → **최소 수정 빠른 적용**. 단, 이상기체만 검증됨.

**권장**: 이 그룹은 Group 3 (APEC) 또는 Group 2 (T-재구성) 방법과 **병렬로 실험**하여 성능 비교에 활용. 단독 사용은 권장하지 않음(실제유체 미지원).

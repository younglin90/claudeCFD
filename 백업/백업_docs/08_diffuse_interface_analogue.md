# Group 8: 확산 계면 스칼라 유사 문제 (참고용)
# (Diffuse Interface Scalar Transport — Abgrall Problem Analogue)

## 위치

이 그룹은 Abgrall 압력 진동 문제의 **구조적 유사체(analogue)** 를 다룬다. 직접적인 압력 진동 해결 방법은 아니지만, 설계 원리가 Abgrall 문제와 동일한 물리적 근거를 가진다.

---

## Paper: Jain & Mani — Stanford CTR (2022) — 계면 한정 스칼라 수송

**파일**: `Jain_surfactant_2022.pdf`
**제목**: "Modeling interface-confined scalars and insoluble surfactants in two-phase flows"
**저널**: Annual Research Briefs 2022, CTR Stanford

### 핵심 문제

비압축성 2상 유동에서 계면에 한정된 스칼라(불용성 계면활성제 `c`)를 수송할 때, 보통의 대류 방정식:
```
∂c/∂t + ∇·(uc) = 0
```
을 쓰면 **스칼라가 계면 밖 벌크 유체로 "누출"(leakage)** 된다.

이것은 Abgrall 문제와 **구조적으로 동일한 수치적 불일치**:
- Abgrall: 보존 에너지 → 압력 역산에서 계면 불일치 → 압력 진동
- 여기: 대류 방정식 → 계면 외부로 스칼라 확산 → 물리적으로 불합리한 분포

### 방법: ACDI 호환 스칼라 수송 + 인공 샤프닝 플럭스

**ACDI 위상장 방정식** (Allen-Cahn 기반):
```
∂ϕ/∂t + ∇·(uϕ) = ∇·{Γ[ε∇ϕ - (1/4)(1 - tanh²(ψ/2ε))·(∇ψ/|∇ψ|)]}
```

**제안하는 스칼라 수송** (계면 한정 보장):
```
∂c/∂t + ∇·(uc) = ∇·{D[∇c - 2(0.5 - ϕ)·n·c/ε]}

여기서:
  D: 인공 확산 계수
  ε: 계면 두께
  n = ∇ψ/|∇ψ|: 계면 법선 벡터
  -2(0.5 - ϕ)·n·c/ε: 인공 샤프닝 플럭스 ← 핵심 항
```

**샤프닝 플럭스의 역할**: `ϕ ≈ 0.5` (계면)에서 소멸, `ϕ → 0` 또는 `ϕ → 1` (벌크)에서 `c`를 계면 쪽으로 밀어냄 → 스칼라를 계면에 한정.

**평형 해** 검증:
```
c_eq ~ 1/cosh²(x/2ε)  ← 계면 델타 함수에 대응
```

**안정성 조건**:
```
Δx ≤ 2D / (|u|_max + Da/2ε)      [공간]
Δt ≤ Δx² / (2·N_d·D)            [시간]
Cell Peclet 조건: Pe_c = Δx|u|_max/D ≤ 1
```

### Abgrall 문제와의 비교

| | Abgrall 압력 진동 | Jain 스칼라 누출 |
|---|---|---|
| **문제 원인** | 계면에서 EOS 역산 불일치 | 계면에서 수송 방정식 불일치 |
| **증상** | 허위 압력 진동 | 스칼라 계면 외부 누출 |
| **해결 원리** | 계면에서 물리 변수의 연속성 이용 | 계면 위치와 스칼라 transport 동기화 |
| **수치적 해결** | 비보존형/PEP/L2-투영 | 인공 샤프닝 플럭스로 계면 한정 |

### 관련성 — 확산 계면 모델(DIM) 사용 시

확산 계면 방법(5방정식 Allaire 모델, Allen-Cahn 기반 CDI 등)을 다성분 압축성 흐름에 적용할 때:

1. **체적분율/위상장 ϕ 의 수송 일관성** — Jain의 샤프닝 플럭스 원리와 동일
2. **종 질량분율 Y_k 의 계면 한정** — 다성분 흐름에서 종이 인접 상으로 누출되지 않도록 설계

Group 2의 Collis 4eq 논문 (`ssrn-5372486.pdf`)에서도 이 문제를 다룬다:
```
상 한정 종 확산 (Eq. 5-7):
dX_α/dt = ... / X_p  [상 몰분율로 정규화 → 크로스 상 확산 방지]
```

## 사용자 코드 적용 권장

**직접 적용 권장하지 않음** — 비압축성 전용.

단, 다음 경우 참고 가치 있음:
- 하이브리드 solver의 비압축성 부분에 계면활성제 모델링이 필요한 경우
- 다성분 흐름에서 종 누출(species leakage) 방지 메커니즘 설계 시 개념적 참고
- Allen-Cahn/CDI 인터페이스 모델과 결합된 다성분 수송 방정식 개발 시

**관련 코드 아이디어**:
ACDI 위상장을 사용하는 경우, 종 수송 방정식에 Jain의 샤프닝 플럭스 원리를 적용하여 크로스-상 종 누출 방지 가능.

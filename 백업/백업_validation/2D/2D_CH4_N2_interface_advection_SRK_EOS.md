# Validation Case — 2D CH4/N2 Interface Advection with SRK EOS

> **출처:** Terashima, Ly & Ihme, JCP 524 (2025), §3.2.2
> **목적:** APEC 스킴의 2D CH4/N2 초임계 블롭 이류 검증; 원형 계면에서 압력 평형 보존 및 에너지 보존 확인

---

## 1. 물리 모델

- **유체:** CH4(메탄) + N2(질소) — 두 성분 초임계 혼합물
- **점성:** 비점성(inviscid)
- **상태방정식:** SRK(Soave-Redlich-Kwong) 실기체 EOS
- **지배방정식:** 비점성 압축성 다성분 Euler 방정식 (2D)

---

## 2. 물성

| 물성 | CH4 (블롭 내부) | N2 (블롭 외부) |
|------|--------------|--------------|
| $\rho$ [kg/m³] | 400.0 | 100.0 |
| $u$ [m/s] | 100.0 | 100.0 |
| $v$ [m/s] | 100.0 | 100.0 |
| $P$ [Pa] | $5 \times 10^6$ | $5 \times 10^6$ |

---

## 3. 계산 도메인 및 메쉬

| 항목 | 값 |
|------|-----|
| 계산 영역 | $[0, 1] \times [0, 1]$ |
| 격자 수 | $N \times N$, $N = 51, 101, 201$ |
| 차원 | 2-D |
| 격자 간격 | 균일(uniform) Cartesian |

---

## 4. 초기 조건

원형 블롭:

$$\rho_{\text{CH4}}(x,y) = \frac{\rho_{\text{CH4},\infty}}{2}\left[1 - \tanh\!\left(k\!\left(\frac{r}{r_c} - 1\right)\right)\right]$$

- $r = \sqrt{(x - x_c)^2 + (y - y_c)^2}$, 블롭 중심 $(x_c, y_c) = (0.5, 0.5)$, 반경 $r_c = 0.25$
- 균일 속도 $(u, v) = (100, 100)$ m/s, 균일 압력 $P = 5 \times 10^6$ Pa

---

| 항목 | 값 |
|------|-----|
| 경계 조건 | Periodic (모든 방향) |
| 최종 시각 | 블롭이 도메인 한 바퀴 순환 후 |
| CFL | 0.3 |

## 8. 참고사항

- 1D 케이스(§3.2.1)의 2D 확장
- 2D에서도 APEC가 FC-NPE보다 현저히 낮은 압력 오차 달성
- 원형 계면 기하에서 방위각 방향 압력 균일성 추가 검증

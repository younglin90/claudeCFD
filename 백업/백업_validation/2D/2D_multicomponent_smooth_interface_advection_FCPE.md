# Validation Case — 2D Multicomponent Smooth Interface Advection (FC-PE Scheme)

> **출처:** Fujiwara et al., JCP 2023, §4.2
> **목적:** 다성분 FC-PE 스킴의 2D 매끄러운 계면 이류 검증; 원형 계면의 압력 평형 및 에너지 보존 동시 달성 확인

---

## 1. 물리 모델

- **유체:** 두 성분(two-component) 이상기체 — 단상(single-phase)
- **점성:** 비점성(inviscid, $\mu = 0$)
- **상태방정식:** 이상기체 (성분별 $\gamma$ 상이)
- **지배방정식:** 비점성 압축성 Euler 방정식 (다성분)

---

## 2. 물성

| 물성 | 내부 (성분 1) | 외부 (성분 2) |
|------|-------------|-------------|
| $\gamma$ [-] | 1.4 | 1.6 |
| $\rho$ [-] | 2.0 | 1.0 |
| $u$ [-] | 1.0 | 1.0 |
| $v$ [-] | 1.0 | 1.0 |
| $P$ [-] | 1.0 | 1.0 |

---

## 3. 계산 도메인 및 메쉬

| 항목 | 값 |
|------|-----|
| 계산 영역 | $[0, 1] \times [0, 1]$ |
| 격자 수 | $N \times N$, $N = 32, 64, 128$ |
| 차원 | 2-D |
| 격자 간격 | 균일(uniform) Cartesian |

---

## 4. 초기 조건

원형 계면의 매끄러운 성분 분포:

$$Y_1(x,y) = \frac{1}{2}\left[1 - \tanh\!\left(\frac{r - r_c}{\delta}\right)\right]$$

- $r = \sqrt{(x-0.5)^2 + (y-0.5)^2}$, $r_c = 0.2$, $\delta$: 계면 두께
- 균일 속도 $(u,v) = (1.0, 1.0)$, 균일 압력 $P = 1.0$

---

## 5. 경계 조건 및 최종 시각

| 항목 | 값 |
|------|-----|
| 경계 조건 | Periodic (주기) |
| 최종 시각 | $t = 1.0$ (계면이 도메인 한 바퀴 순환) |

---

## 6. 수치 설정

| 항목 | 값 |
|------|-----|
| 공간 스킴 | FC-PE scheme (Fujiwara et al.) |
| 비교 스킴 | FC-only, 표준 보존 방법 |
| 시간 적분 | SSP-RK3 |
| 시간 CFL | 0.5 |

---

## 7. 출력 변수 및 결과 비교

### 7.1 오차 분포

| 측정값 | 기대 결과 |
|--------|---------|
| $\max|P - P_0|$ | FC-PE scheme에서 거의 0 |
| $\max|u - u_0|$ | FC-PE scheme에서 거의 0 |
| 에너지 보존 오차 | 기계 정밀도 |

### 7.2 수렴 분석

- 격자 수 증가에 따른 압력/속도 오차 수렴 확인
- FC-PE vs FC-only 비교: 압력 오차 크기 비교

---

## 8. 참고사항

- 2D 원형 계면 이류는 1D 케이스의 다차원 확장
- 원형 계면에서 압력 진동이 발생하면 가짜 충격파 생성 가능
- FC-PE scheme의 2D 구현에서 교차 미분항(cross-derivative) 처리 검증

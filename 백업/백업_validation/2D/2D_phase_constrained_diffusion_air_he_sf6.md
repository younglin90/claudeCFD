# Validation Case — 2D Phase-Constrained Diffusion: Air-Helium-SF6

> **출처:** Collis et al., preprint (2025), §4.3.2
> **목적:** 다상 다성분 유동에서 위상 제약 확산(phase-constrained diffusion) 검증; 불혼화성 기-액 계면을 가로지르는 성분 누설 방지 확인

---

## 1. 물리 모델

- **유체:** 공기(Air) + 헬륨(He) + SF6 — 두 상(two-phase), 세 성분(three-component)
  - 기상: 공기 + 헬륨 (혼화성)
  - 액상: SF6 (불혼화성)
- **인터페이스:** 불혼화성 기-액 계면에 CDI 정규화 활성
- **확산:** 기상 내 성분 질량 확산 활성; 위상 제약 확산으로 기-액 계면 성분 누설 방지
- **점성:** 점성 활성
- **상태방정식:** NASG EOS

---

## 4. 초기 조건

- 기상(Air + He) 구간과 SF6 액상 구간이 공존하는 혼합 도메인
- 기상 내 공기/헬륨 성분 분포 초기화 (확산 계면 형태)
- 불혼화성 기-액 계면 CDI 초기화

## 7. 출력 변수 및 결과 비교

### 7.1 시각화 비교

| 비교 항목 | 내용 |
|-----------|------|
| 위상 제약 확산 비활성 | 기-액 계면 성분 누설(SF6 → 기상) 발생 |
| 위상 제약 확산 활성 | 기-액 계면 성분 누설 방지 |

### 7.2 검증 기준

- 위상 제약 확산 활성 시 기-액 계면 성분 누설 $\approx 0$
- 기상 내 Air-He 확산은 물리적으로 올바르게 진행
- CDI 정규화로 불혼화성 계면 두께 일정 유지

---

## 8. 참고사항

- 위상 제약 확산(phase-constrained diffusion): 기-액 계면에서 성분이 다른 상으로 이동하지 않도록 확산 플럭스를 제약하는 기법
- 이 기법 없이는 WENO 스킴 사용 시에도 수치 확산으로 SF6가 기상으로 누설됨
- 실제 응용: 수중 폭발, 캐비테이션, ICF 혼합층 문제

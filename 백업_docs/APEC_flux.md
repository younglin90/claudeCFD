#### 핵심 아이디어

에너지 보존(full conservation)과 압력평형(PE) 보존을 **동시에** 근사적으로 만족하는 수치 플럭스 구성.  
기존 FC-NPE(표준 split-form)는 에너지는 보존하지만 PE를 위반 → 발산.  
Fujiwara [2023]은 PE를 정확히 만족하지만 calorically perfect gas + 특정 혼합규칙에만 적용 가능.  
**APEC는 임의의 EOS(SRK 포함)에 적용 가능한 첫 번째 완전보존 + 근사 PE 보존 기법.**

---

#### 압력평형(PE) 조건 유도

균일 압력·속도 초기조건 (`∂ₜp = 0`)에서 에너지 방정식을 분해하면 flux에 대한 PE 조건이 유도됨:
```
∂ₓF_ρE = Σᵢ εᵢ · ∂ₓF_ρYᵢ + u · ∂ₓF_ρu

εᵢ = (∂ρe/∂ρᵢ)_{ρⱼ≠ᵢ, p}
```

이를 이산화하면 half-point 값에 대한 조건:
```
ρe|_{m+1/2} - ρe|_{m-1/2} = Σᵢ εᵢ|_m · (ρYᵢ|_{m+1/2} - ρYᵢ|_{m-1/2})
```

→ εᵢ가 셀 m에서 평가되어 **비대칭**이므로, 이를 정확히 만족하는 locally-defined half-point 값은 존재하지 않음.  
→ APEC는 이를 **대칭화하여 근사적으로** 만족시킴.

---

#### APEC Half-point 값 (핵심 수식)

종밀도 half-point (단순 평균):
```
ρYᵢ|_{m+1/2} = (ρYᵢ|_m + ρYᵢ|_{m+1}) / 2
```

내부에너지 half-point (**APEC의 핵심**, PE 보정항 포함):
```
ρe|_{m+1/2} = (ρe|_m + ρe|_{m+1}) / 2
            - Σᵢ [(εᵢ|_{m+1} - εᵢ|_m) / 2] · [(ρYᵢ|_{m+1} - ρYᵢ|_m) / 2]
```

표준 기법 대비 추가된 **보정항** `- Σᵢ Δεᵢ/2 · ΔρYᵢ/2` 이 PE 오차를 감소시킴.

---

#### APEC 수치 플럭스 전체 (1D split-form)
```
ρYᵢu|_{m+1/2}     = [(ρYᵢ|_m + ρYᵢ|_{m+1})/2] · [(u|_m + u|_{m+1})/2]

ρuu|_{m+1/2}       = [(ρ|_m + ρ|_{m+1})/2] · [(u|_m + u|_{m+1})/2]²

p|_{m+1/2}         = (p|_m + p|_{m+1}) / 2

ρu²/2 · u|_{m+1/2} = [(ρ|_m + ρ|_{m+1})/2] · [u|_m·u|_{m+1}/2] · [(u|_m + u|_{m+1})/2]

pu|_{m+1/2}        = (p|_{m+1}·u|_m + p|_m·u|_{m+1}) / 2

ρeu|_{m+1/2}       = ρe|_{m+1/2} · [(u|_m + u|_{m+1})/2]   ← APEC half-point 사용
```

총 에너지 플럭스: `F_ρE = ρeu|_{m+1/2} + ρu²/2·u|_{m+1/2} + pu|_{m+1/2}`

#### PE 오차 분석 (modified equation)

| 기법 | PE 오차 선두항 계수 | 차수 |
|------|-------------------|------|
| FC-NPE (표준) | `1/3 · ∂εᵢ/∂x · ∂²ρYᵢ/∂x²` + `1/6 · ∂²εᵢ/∂x² · ∂ρYᵢ/∂x` | O(Δx²) |
| **APEC** | `1/12 · ∂εᵢ/∂x · ∂²ρYᵢ/∂x²` - `1/12 · ∂²εᵢ/∂x² · ∂ρYᵢ/∂x` | O(Δx²) |

→ 계수가 **최대 4배 작음** → PE 오차 최대 1/4 수준으로 감소 (수치 검증 일치)

#### εᵢ 계산 (SRK EOS 적용 시)
```
εᵢ = (∂ρe/∂ρᵢ)_{ρⱼ≠ᵢ, p}
   = -(ρCᵥ / (∂p/∂T)_ρᵢ) · (∂p/∂ρᵢ)_{ρⱼ≠ᵢ,T} + (∂ρe/∂ρᵢ)_{ρⱼ≠ᵢ,T}
```

SRK EOS에 대한 편미분 `(∂p/∂T)`, `(∂p/∂ρᵢ)`, `(∂ρe/∂ρᵢ)_T` 는 논문 Eq. (49)~(51) 참조.

#### upwind 기법(HLLC 등)으로 확장 시 (Appendix A)
```
F_ρE|_{m+1/2} = (F_ρE|_m + F_ρE|_{m+1}) / 2
              + (1/2) Σᵢ (εᵢ - u²/2)|_m   · (F_ρYᵢ|_{m+1/2} - F_ρYᵢ|_m)
              + (1/2) u|_m                 · (F_ρu|_{m+1/2}  - F_ρu|_m)
              - (1/2) Σᵢ (εᵢ - u²/2)|_{m+1} · (F_ρYᵢ|_{m+1} - F_ρYᵢ|_{m+1/2})
              - (1/2) u|_{m+1}              · (F_ρu|_{m+1}   - F_ρu|_{m+1/2})
```

`F_ρYᵢ`, `F_ρu` 는 선택한 upwind 기법(HLLC 등)으로 계산.

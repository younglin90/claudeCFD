# Validation Case — 1D Multiphase Multicomponent Shu-Osher Problem

> **출처:** Collis et al., preprint (2025), §4.3.1
> **목적:** 다상 다성분 유동(multiphase + multicomponent)에서 CDI 정규화와 위상 제약 확산(phase-constrained diffusion)의 결합 적용 검증; 충격파와 성분 계면 동시 포착

---

## 1. 물리 모델

- **유체:** 두 상(two-phase) + 두 성분(two-component) — 불혼화성 기-액 + 기상 내 혼화성 다성분
- **인터페이스:** 불혼화성 계면에 CDI 정규화 활성; 혼화성 성분 간 확산 활성
- **점성:** 비점성(inviscid, $\mu = 0$)
- **상태방정식:** NASG EOS (Noble-Abel Stiffened Gas)
- **지배방정식:** 비점성 압축성 Euler 방정식

---

## 2. 물성

| 물성 | 성분 1 (고압 기상) | 성분 2 (저압 기상) | 액상 |
|------|-------------------|-------------------|------|
| $\gamma$ [-] | 1.4 | 1.67 | 5.5 |
| $P^\infty$ [-] | 0 | 0 | 1.505 |

> 구체적인 초기 밀도·압력값은 표준 Shu-Osher 문제에서 변형; 논문 내 설정 참조

---

## 3. 계산 도메인 및 메쉬

| 항목 | 값 |
|------|-----|
| 계산 영역 | $x \in [-5, 5]$ (또는 논문 설정 참조) |
| 격자 수 | $N = 400$ (또는 논문 설정 참조) |
| 차원 | 1-D |
| 격자 간격 | 균일(uniform) Cartesian |

---

## 4. 초기 조건

- Shu-Osher 문제 변형: 고압 기상 충격파 + 기상 내 밀도 진동 + 불혼화성 액상 구간 포함
- 충격파: 좌측 고압 상태, 우측 저압 상태 + 주기적 밀도 진동
- 불혼화성 액상 구간: 우측 도메인 일부에 액적으로 배치

---

## 5. 경계 조건 및 최종 시각

| 항목 | 값 |
|------|-----|
| 경계 조건 | Transmissive (비반사) |
| 최종 시각 | 충격파가 진동 구간을 통과한 후 |

---

## 6. 수치 설정

| 항목 | 값 |
|------|-----|
| 공간 스킴 | WENO5JS, WENO5Z, TENO6 |
| Riemann solver | HLLC |
| 시간 적분 | SSP-RK3 |
| 시간 CFL | 0.5 |

---

## 7. 출력 변수 및 결과 비교

### 7.1 공간 분포

| 그래프 | 변수 |
|--------|------|
| (a) | 밀도 $\rho$ |
| (b) | 속도 $u$ |
| (c) | 압력 $P$ |
| (d) | 체적분율 $\phi$ |
| (e) | 성분 질량분율 $Y_s$ |

### 7.2 검증 기준

- 충격파, 밀도 진동, 불혼화성 계면 동시 포착
- CDI 정규화로 계면 두께 일정 유지
- 위상 제약 확산으로 기-액 계면 간 성분 누설 방지
- 기준 해(fine-grid)와의 비교를 통한 수렴성 확인

---

## 8. 참고사항

- 다상 다성분 조합 문제: 불혼화성 상 + 혼화성 성분의 공존
- 위상 제약 확산(phase-constrained diffusion) 없이는 계면에서 성분 누설 발생
- 이 케이스는 실제 ICF/폭발 문제의 단순화 모델

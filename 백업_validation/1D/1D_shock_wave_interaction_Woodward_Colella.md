# Validation Case — 1D Shock Wave Interaction (Woodward-Colella, ACID)

> **출처:** Denner et al., JCP 367 (2018), §7.4.2
> **참조:** Woodward & Colella, *Journal of Computational Physics* 54 (1984) 115–173
> **목적:** 두 충격파의 상호작용(Woodward-Colella blast wave) 검증; ACID 스킴의 강한 충격파 포착 성능 확인

---

## 1. 물리 모델

- **유체:** 단일 이상기체 (단상)
- **점성:** 비점성(inviscid)
- **상태방정식:** 이상기체 ($\gamma = 1.4$)
- **지배방정식:** 압축성 Euler 방정식

---

## 2. 물성

| 물성 | 값 |
|------|-----|
| $\gamma$ [-] | 1.4 |
| $P^\infty$ [-] | 0 |

---

## 3. 계산 도메인 및 메쉬

| 항목 | 값 |
|------|-----|
| 계산 영역 | $x \in [0, 1]$ |
| 격자 수 | $N = 400$ (또는 여러 해상도) |
| 차원 | 1-D |

---

## 4. 초기 조건 (Blast Wave)

$$\text{좌측 } (0 < x < 0.1): \quad (\rho, u, P) = (1, 0, 1000)$$
$$\text{중간 } (0.1 \leq x \leq 0.9): \quad (\rho, u, P) = (1, 0, 0.01)$$
$$\text{우측 } (0.9 < x < 1): \quad (\rho, u, P) = (1, 0, 100)$$

---

## 5. 경계 조건 및 최종 시각

| 항목 | 값 |
|------|-----|
| 경계 조건 | 반사(벽) — 좌우 양쪽 |
| 최종 시각 | $t = 0.038$ |

---

## 6. 수치 설정

| 항목 | 값 |
|------|-----|
| 공간 스킴 | ACID |
| 시간 적분 | Runge-Kutta |
| 시간 CFL | 0.5 |

---

## 7. 출력 변수 및 결과 비교

| 그래프 | 변수 |
|--------|------|
| (a) | 밀도 $\rho$ |
| (b) | 속도 $u$ |
| (c) | 압력 $P$ |

**비교 대상:** 초고해상도($N = 4000$) 기준 해 (Woodward & Colella, 1984)

---

## 8. 참고사항

- Woodward-Colella blast wave: 두 강한 충격파의 반사와 상호작용
- 매우 높은 압력비($P_{\max}/P_{\min} = 10^5$) — 수치 방법의 극한 테스트
- 이 케이스에서 ACID 스킴의 강한 충격파 포착 성능 검증
